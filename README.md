# WPS BLOCKS
A set of gutenberg blocks for WordPress.
Contact info\
Grid\
Section\
Slider\
Image Slider (deprecated)\
Media Slider\
Whatsapp\
Card

### Requirements
NPM >= 18.0.0

PHP >= 8.0.0

WordPress >= 6.0.2

### Installation
Clone this repo into your plugins directory

```npm install && npm run build```
